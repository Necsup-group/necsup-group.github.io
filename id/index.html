Necsup ID

Almost too simple

Necsup ID is set up to be the most seamless way to connect you and every other user to your favorite Necsup apps. Once you login to your ID on a device, you'll have full access to every Necsup app on that device for a limited time or until you sign out.

Security first

We at Necsup know that for a platform and ecosystem like this to be successful, security needs to be the core foundation. Therefore, we always focus on implementing the best ways to secure all of our users' data in Necsup ID and every other app.

Necsup ID is currently secured by keeping devices signed in for only 1 month maximum. This balances security and convenience, reducing the friction of constant 2FA codes while ensuring your sensitive information remains protected.
